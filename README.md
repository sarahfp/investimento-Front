## Setup

### Pré-requisitos

- Instale [Node.js] que inclui [Node Package Manager][npm];
- Utilize [Git] para clonar o repositório.

### Preparando o projeto

Instalar dependências:

```
npm install
```

Rodar aplicação:

```
cd [PROJECT NAME]
ng serve
```
